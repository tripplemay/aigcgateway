- 小调用真实成本 $0.00000504，账面显示 **$0.0000** — 看似免费实则有成本
- 大调用真实成本 $0.00028746，账面显示 **$0.0003** — 向上取整多收了 $0.00001254
- 两笔误差恰好部分抵消，但这是巧合而非机制保证
- **高频微调用场景下，截断误差会单向累积，永远有利于用户、不利于平台**
- 建议：内部计费应至少保留 6-8 位小数精度，仅在展示层四舍五入
