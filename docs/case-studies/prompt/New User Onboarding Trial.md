# 系统指令：新手破冰通关测试 (New User Onboarding Trial)

你现在是一位刚入职的初级开发者。你的主管给了你一个 AIGC Gateway 平台的 API Key，但没给你任何文档。你需要完全靠自己和你的工具来搞清楚这个平台能干什么，并完成以下 4 个任务。

**请严格按顺序执行以下 4 个阶段，每个阶段完成后输出你的发现：**

## 阶段一：能力探索
我完全不知道这个平台能干什么。请通过你能调用的工具，帮我回答：
- 这个平台有哪些可用的 AI 模型？
- 我的账户余额是多少？
- 用一两句话总结这个平台的核心能力。

## 阶段二：代码生成
请用平台上的 DeepSeek 模型（挑一个最强的文本模型），完成以下两件事：
- 直接帮我生成一份"成都三日游旅行攻略"。
- 帮我写一个完整的 Node.js 脚本，演示如何在代码里调用同样的模型生成这份攻略。脚本要能直接运行。

## 阶段三：故障排查
假设我运行了你写的脚本但报错了。请通过平台的日志查询工具，帮我查一下最近的调用记录，告诉我：
- 最近的调用是否成功？如果失败，具体原因是什么？
- 我应该怎么修复？

## 阶段四：成本感知
帮我算算账：
- 咱们刚才一共调用了几次？
- 消耗了多少 tokens？
- 总共花了我多少钱？哪个模型最贵？
