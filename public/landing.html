FAQ · 常见问题

你可能会问的，我们想在前面

下面是接入前最常被问到的 5 个问题，如果还没解答你的疑问，可以点右上角文档或加入社群。

定价是怎么算的？有没有溢价或服务费？
预充值模式，按 token 用量扣费，与各服务商官网定价一致，不额外加价。人民币计价，控制台实时可见每次调用的成本。
支持哪些模型？新模型多久能上？
覆盖 11 家主流服务商的 100+ 模型（文本对话、推理、图像生成、向量、多模态）。新模型通常在官方发布后 1-3 天内同步上架，控制台的"模型管理"页可实时查看当前可用模型清单。
我的 Prompt 和输出数据是否会被用于训练？
不会。传输链路全程加密，审计日志仅项目所属账号可见，平台方不会将你的业务数据用于模型训练或二次分发。服务商侧数据策略以各家官方条款为准。
某家服务商挂了怎么办？会影响我的应用吗？
平台运行自动健康检查，单家服务商异常时会自动切换到同类能力的其他 channel（failover 策略可配置），你的应用侧不需要任何代码改动，控制台的"调用日志"会标注每次切换。
停服 / 迁移是否方便？会被锁定吗？
不会。协议 100% 兼容 OpenAI SDK，任何时候你都可以把 base_url 切回原厂直连，没有私有协议绑定。Template / Action 支持 JSON 导出，便于迁移或备份。
