## 开发时主要查阅

- `docs/specs/` — 实现规格文档（API、数据库、服务商适配、支付、控制台交互、SDK、部署等）
- `docs/provider/` — 各服务商接入 ADR（新增服务商时参考）
- `docs/design-draft/` — Stitch UI 设计稿 HTML（页面还原时参考）

## 测试相关

- `docs/test-cases/mcp-integration-test-cases.md` — MCP 集成测试用例（含 [L1]/[L2] 层级标注）
- `docs/test-reports/` — 测试签收报告（Evaluator 写入）
- `AGENTS.md` — Codex/Agent 行为规范，含 §17 分层测试策略

## 不需要主动阅读

- `docs/archive/` — 历史文档
- `docs/test-cases/` 和 `docs/test-reports/` — 开发时不需要主动读取
