# Balance — Design Notes

## Ignore (fabricated)
- "Estimated Expiry ~12 days" — no burn-rate calculation API

## Partial support
- Alert threshold save — PATCH /projects/:id supports alertThreshold value, but no notification delivery mechanism exists
- Alert via email/SMS — threshold stored, no sending mechanism

## Fully supported
- Balance display, last recharge, transaction history (with type filter + pagination), recharge button
