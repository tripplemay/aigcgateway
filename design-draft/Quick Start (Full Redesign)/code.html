Quick Start Redesign

Initialize your journey into automated creativity with our streamlined SDK integration. Four steps to production-ready AI.

1

Install SDK

Environment

Prepare your local environment by installing the AIGC core gateway package via NPM or Yarn.

Terminal 
npm install @aigc-gateway/core
# or
yarn add @aigc-gateway/core
2

Basic Chat

Foundation

Initialize the client and send your first synchronous prompt to the default gateway.

index.js 
const gateway = new Gateway({ apiKey: 'YOUR_KEY' });

const response = await gateway.chat.create({
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'Hello!' }]
});
3

Streaming Response

Real-time

Enable low-latency interfaces by streaming tokens directly to your client side.

stream.js 
const stream = await gateway.chat.create({
  model: 'gpt-4',
  stream: true,
  messages: [{ role: 'user', content: 'Tell a story' }]
});

for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0].delta.content);
}
4

Image Generation

Creative

Connect to Midjourney or DALL-E endpoints for high-fidelity visual synthesis.

images.js 
const image = await gateway.images.generate({
  prompt: 'A futuristic laboratory',
  n: 1,
  size: '1024x1024'
});

console.log(image.data[0].url);
