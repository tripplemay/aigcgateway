# Actions List — Design Notes

## Ignore (fabricated)
- Semantic version "v2.4.1" format — versionNumber is Int (1,2,3). Display as "v1", "v2", "v3".

## Partial support
- "Variables" count column — GET /actions list response omits variables. Can add to API or derive from detail call. For now omit this column or show dash.

## Fully supported
- Action name, model, description, updated date, search, pagination, "Create Action" button
